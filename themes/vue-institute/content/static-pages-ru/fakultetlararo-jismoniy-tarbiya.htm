“Чирчикский государственный педагогический институт был образован по приказу ректора в 2019 году 10-октября 01-660т-число.В 2019 году 27-августа А.Р.Абдулахатов в научном собрании по первому приказу становится заместителем кафедры Чирчикского государственного педагогического института.

Учителя профессоры кафедры дают уроки по данным предметам,а также проводят научные работы.

 

Проводимые предметы межфакультетного физического воспитания.

С сегодняшнего дня в кафедре имеются учителя профессоры по разным научным направлениям.

                     В состав кафедры входят учителя, которые ведут свою деятельность.

 

  • Абдулахатов Акром Розматович- заведующый кафедры,доцент,специалист по методике «Виды борьбы».

Маматкулов Равшанжон Салижанович-учитель спортивной деятельности специалист по волейболу и мастер спорта. 

  • Жумаев Абдухаким Турдиевич-учитель по спортивной деятельности,специалист по волейболу,мастер спорта.
  • Юсупбаева Омонгул Сапарбаевна-учительница по основам валеологии,специалист.
  • Умаралиева Дильфуза Умарали кизи-учительница по лёгкой атлетике,специалист,учитель-тренер высшей категории.
  • Худойберганов Жавлонбек Саотбаевич-учитель по борьбе,специалист.
  • Чтобы провести упражнения,кроме учителей кафедры,связанные с направлением «физическая культура и спорт» специалист профессор по этому направлению проводит трудовую деятельность. 

 

 

В кафедре есть 2  спорт зала,здесь проводят уроки,а также проводятся кружки.В зале имеются все оборудования.Профессоры кафедры проводят разные кружки,а также проводят соревнования,чтобы ученики могли показать свои навыки и достичь определённых целей.

              

 

 

Чтобы улучшить учебный процесс профессры-учителя пользуются современными учениями,способы мультимедии и коммуникационной технологией.

 

         Кафедра “Ташкентская область город Чирчик 24-общеобразовательная школа установлены дружеские связи с другими организациями.

Международные дружеские связи.

 


В сегодняшний день группа кафедры установила связи и начала вести дружеские работы с иностранными институтами и университетами.Южный Казахстанский государственный институт имени Мухтара Авезова планируют прповести научные работы.
